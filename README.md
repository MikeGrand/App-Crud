Este proyecto es la primera parte de un sistema de registro multiplataforma, enfocado en el desarrollo de una API REST, consumida por una aplicación web. El sistema permite crear, listar, modificar y eliminar registros utilizando una arquitectura basada en API, lo que facilita su reutilización en futuras app móviles.

Este proyecto corresponde a la **Parte 1** del sistema completo:
- Parte 2: Aplicación de escritorio
- Parte 3: Aplicación móvil

Características

- Registro de usuarios
- Validación de datos
- Listado dinámico desde API
- API en PHP con respuestas JSON
- Prevención básica de XSS (`htmlspecialchars`)
- Diseño web con Bootstrap
- Arquitectura separada (Frontend / API)
